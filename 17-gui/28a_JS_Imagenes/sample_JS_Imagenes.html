Imagenes

Imagen disponible en 960(alto) x 1280(ancho) presentada con un ancho de 800

Los valores de tamaño de la imagen se almacenan en diversas propiedades del nodo DOM:

  • height: 600
  • width: 800
  • clientHeight: 600
  • clientWidth: 800
  • naturalHeight: 960 - resolución real
  • naturalWidth: 1280 - resolución real
  • scrollHeight: 600
  • scrollWidth: 800

También es posible almacenar una imagen como un objeto Imagen creado en JS y vinculado con un fichero determinado:

var imgRoyo = new Image();
imgRoyo.src = "205356_luis_royo.jpg"

Esta imagen pre-cargada puede ser incrporada desde JS a una etiqueta IMG,
en este caso con su tamaño al 50% calculado a partir del valor del objeto Imagen

var domImg2 = document.getElementById("lroyo2");
domImg2.src = imgRoyo.src;
domImg2.alt = "Ilustración de Luis Royo";
imgRoyo.onload = function () {
domImg2.width = imgRoyo.width / 2;
}
