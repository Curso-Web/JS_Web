Página HTML para manipular el DOM

Aqui comienza un parrafo que podemos manipular facilmente usando

parrafos = document.getElementsByTagName("p");
alert (parrafos[0])

Si tienes firebug, veras en la consola que se ha creado un HTMLCollection[p, p, p]
al que puedes acceder directamente o desde la pestaña DOM-window
para ccomprobar el contenido recogido en el array especial de objetos (parrafos)

alert(parrafos[0].textContent)

Ahora desde la consola puedes cambiar completamente este parrafo

Aqui tenemos un segundo parrafo en el que el atributo name toma el valor de "especial"
Es un atributo invisible que normalmente se incluye en los items de un formulario,
pero tambien se puede usar con parrafos, como hemos hecho aqui.
Para manipular este parraso podemos usar

parrafoEspecial = document.getElementsByName("especial")
alert (parrafoEspecial)

Si tienes firebug, veras en la consola que se ha creado un NodeList[p]
al que puedes acceder directamente o desde la pestaña DOM-window
para ccomprobar el contenido recogido en el objeto de tipo NodeList

alert(parrafoEspecial[0].textContent)

Ahora desde la consola puedes cambiar completamente este parrafo

Aqui tenemos otro parrafo "especial"

Aqui tenemos un último parrafo, y en este caso es el atributo id el que va a tomar el valor de "final"
Para manipular este parraso podemos usar

parrafoFinal = document.getElementById("final")
alert (parrafoFinal)

Si tienes firebug, veras que en la consola aparece directament el Nodo[p] en cuestión
al que puedes acceder directamente o desde la pestaña DOM-window
para ccomprobar el contenido recogido en el objeto de tipo Nodo

alert(parrafoFinal.textContent)

Ahora desde la consola puedes cambiar completamente este parrafo
